# JWT (JSON Web Token)  

## 1. JWT(JSON Web Token)란?  
- 웹에서 사용되는 <u>JSON 형식의 토큰에 대한 표준 규격</u>  
- 사용자의 인증(authentication) 또는 인가(authorization) 정보를 서버와 클라이언트 간에 안전하게 주고 받기 위해서 사용
- 토큰 기본 - Stateless (쿠키 & 세션과 비교)/ 다른 토큰과 비교 - Claim 기반
